A distributed, full-stack intelligence platform designed to cross-reference unidentified forensic remains with missing persons registries using advanced AI object detection and biometric probability matrices.

Explore the Repository View System Architecture

Engineered for Scale & Security

🧠

AI Microservice Pipeline

Images are routed from the Node.js backend to an isolated Python service. YOLOv8 object detection isolates artifacts, while regression models estimate biological parameters.

🔐

Strict Data Lifecycle

Protected by JWT Authentication. Employs a robust Soft-Delete architecture, moving closed cases to a Cold Storage Recycle Bin to ensure forensic data provenance.

⚡

Probability Matrix

A custom SQL-driven algorithm that dynamically calculates match confidence scores based on geospatial proximity, age/height overlaps, and extracted item arrays.
